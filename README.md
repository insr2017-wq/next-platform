# next-platform

Modern web platform skeleton built with **Next.js App Router**.

## Getting started

1. Install Node.js (LTS) and npm.
2. Install deps:
   - `npm install`
3. Run dev server:
   - `npm run dev`

## Routes included

User routes:
- `/`
- `/login`
- `/register`
- `/dashboard`
- `/deposit`
- `/withdraw`
- `/invite`
- `/profile`
- `/my-products`

Admin routes:
- `/admin`
- `/admin/dashboard`
- `/admin/users`
- `/admin/transactions`
- `/admin/withdrawals`
